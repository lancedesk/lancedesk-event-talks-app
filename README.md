# Tech Talks Today

This project is a single-page website for a 1-day tech talks event.

## Features

*   **Schedule:** View the entire day's schedule with timings for each talk.
*   **Search:** Search for talks by category.
*   **Expandable Details:** Click on a talk to view its full description.
*   **Serverless:** The website is a single, serverless HTML file.

## How to run

1.  Clone the repository.
2.  Open the `dist/index.html` file in your browser.

Alternatively, you can run a simple web server:

1.  Navigate to the `dist` directory.
2.  Run `python3 -m http.server`.
3.  Open your browser and go to `http://localhost:8000`.
